有兴趣的可以去瞧瞧。这位博主分析的很全，小生受教了

由于Github是国外的服务器，而上面存储图片的服务器也是在国外，所以国内的用户有时会出现图片不能显示的情况，这可能是

  1. dns被污染了
  2. host设置错误
  3. 官方更新了dns，而用户电脑上没有同步更新dns导致不同步所发生的错误
